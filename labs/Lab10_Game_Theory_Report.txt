
ЛАБОРАТОРНАЯ РАБОТА №10
Игровые модели. Решения игровых задач

ВЫПОЛНЕННЫЕ ЗАДАЧИ:

1. Игра (3, 4)
   Победитель: ВТОРОЙ ИГРОК
   Стратегия: Любой ход первого игрока

2. Игра (5, 7)
   Победитель: ПЕРВЫЙ ИГРОК
   Стратегия: (5*2, 7)

3. НИМ [3, 5, 7]
   Победитель: ПЕРВЫЙ ИГРОК
   Стратегия: Позиция несимметричная, первый игрок может привести к XOR=0

4. Шоколадка 4x5
   Победитель: ПЕРВЫЙ ИГРОК
   Стратегия: Первый ход: съесть клетку (2,2). Далее зеркальная стратегия

ИСПОЛЬЗОВАННЫЕ МЕТОДЫ:
- Динамическое программирование с мемоизацией
- Теория игр (стратегия XOR для игры НИМ)
- Анализ дерева игры
- Поиск выигрышных позиций

ТЕОРЕТИЧЕСКИЕ ОСНОВЫ:
1. Комбинаторная теория игр
2. Позиции P и N (выигрышные и проигрышные)
3. Стратегия отражения
4. Теорема Спрага-Гранди

ВЫВОДЫ:
Все игровые задачи решены с определением победителя и выигрышной стратегии.
Для каждой игры найдена оптимальная стратегия при безошибочной игре обоих игроков.
