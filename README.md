# Лабораторные работы №5-10 по курсу "Информационные технологии"

## Описание

Данный репозиторий содержит выполненные лабораторные работы №5-10 по курсу "Информационные технологии". 
Все работы реализованы с использованием современных Python библиотек для автоматической генерации документов, таблиц, презентаций и решения задач.

## Структура проекта

```
labaaa/
├── labs/                           # Директория с лабораторными работами
│   ├── lab5_word_formatting.py    # Лабораторная №5 (Word - форматирование)
│   ├── lab6_word_tables.py        # Лабораторная №6 (Word - таблицы)
│   ├── lab7_part1_salary.py       # Лабораторная №7.1 (Excel - зарплата)
│   ├── lab7_part2_graphs.py       # Лабораторная №7.2 (Excel - графики)
│   ├── lab7_part3_sorting.py      # Лабораторная №7.3 (Excel - сортировка)
│   ├── lab8_powerpoint.py         # Лабораторная №8 (PowerPoint)
│   ├── lab9_graph_problems.py     # Лабораторная №9 (Графы)
│   └── lab10_game_theory.py       # Лабораторная №10 (Теория игр)
├── run_all_labs.py                # Мастер-скрипт для запуска всех работ
└── README.md                      # Этот файл
```

## Использованные технологии

### Python библиотеки:
- **python-docx** - создание и редактирование документов Word (.docx)
- **openpyxl** - работа с файлами Excel (.xlsx), формулы, диаграммы
- **python-pptx** - создание презентаций PowerPoint (.pptx)
- **networkx** - работа с графами, алгоритмы поиска путей
- **matplotlib** - визуализация графов и построение графиков

### Реализованные алгоритмы:
- Поиск кратчайших путей в графах
- Динамическое программирование с мемоизацией
- Теория игр (стратегия XOR для НИМ)
- Анализ дерева игры

## Особенности реализации

### Почему Python вместо ручной работы в Office?

1. **Автоматизация:** Все лабораторные работы выполняются автоматически одним запуском
2. **Воспроизводимость:** Можно запустить скрипты в любое время и получить тот же результат
3. **Современные технологии:** Использование актуальных библиотек Python
4. **Кросс-платформенность:** Работает на Windows, Linux, macOS
5. **Легко модифицировать:** Можно быстро изменить данные или параметры

### Преимущества автоматизации:

- ✅ Нет необходимости в установке Microsoft Office
- ✅ Все формулы и вычисления выполняются программно
- ✅ Легко добавить новые данные или изменить существующие
- ✅ Результаты полностью воспроизводимы
- ✅ Код можно использовать как шаблон для других задач
